# Перевірка проходження цеглини у вікно

### Короткий опис
Ця програма вирішує задачу проходження **прямокутного паралелепіпеда** (цеглини, що має три виміри: $a, b, c$) через **прямокутний отвір** (вікно, що має два виміри: $x, y$). Програма перевіряє **всі три можливі орієнтації** цеглини і визначає, чи можливо хоча б однією гранню пронести об'єкт через отвір.

###  Технології та мова
* **Мова:** C++
* **Технологія:** Умовна логіка та перевірка перестановок (шляхом сортування розмірів за допомогою `std::swap`).
* **Структура даних:** Прості типи даних (`double`, `bool`).

### Як запустити програму

1.  **Компіляція:** 
    ```bash
    g++ brick_path.cpp -o brick_runner
    ```
2.  **Запуск:** 
    ```bash
    ./brick_runner
    ```
